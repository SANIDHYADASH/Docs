
# Watermelon Wars

## Game Overview

**Watermelon Wars** is a fast-paced, multiplayer party game built with React, Vite, TypeScript, shadcn-ui, and Tailwind CSS. Compete with friends in fruity arenas, test your reflexes and memory, and climb the leaderboard to become the ultimate Watermelon Champion!

## Game Stages & Objectives

### Stage 1: Watermelon Rush
- **Objective:** Tap or click on watermelons as they appear to collect points.
- **Scoring:**
  - Common Watermelon: 10 points
  - Rare Watermelon: 25 points
  - Epic Watermelon: 50 points
  - Legendary Watermelon: 100 points
- **Power-ups:** Speed boosts, shields, score multipliers spawn randomly.
- **Obstacles:** Bombs and rotten fruits (avoid them!).
- **Tips:** The faster you collect, the higher your score. Race against the clock and compete in real-time with other players.

### Stage 2: Melon Puzzle Match
- **Objective:** Flip tiles to reveal fruit and match pairs to score points.
- **Scoring:**
  - Successful Match: 50 points
  - Special Tiles:
	 - Golden: 150 points
	 - Mystery: 75-200 points
	 - Bomb: 100 points (reveals all tiles briefly)
	 - Time: 80 points
- **Obstacles:** Time pressure and tricky patterns.
- **Tips:** Use Bomb tiles strategically to memorize positions. Find the most matches before time runs out.

## Multiplayer & Scoring
- Compete in real-time rooms with friends or random players.
- Leaderboard updates live as players score points.
- Each player has a unique watermelon color and name.
- The winner is the player with the highest score at the end of the game timer.

## Pro Tips & Strategy
- **Prioritize High-Value Targets:** Focus on rare, epic, and legendary watermelons in Stage 1 for maximum points.
- **Use Special Tiles Wisely:** In Stage 2, Bomb tiles reveal all tiles temporarily—use them to memorize multiple positions.
- **Stay Focused:** React quickly; every second counts in this multiplayer competition.
- **Practice Makes Perfect:** Play multiple rounds to master both stages and improve your reflexes and memory.

## Technologies Used
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How to Play Locally

1. **Clone the repository:**
	```sh
	git clone <YOUR_GIT_URL>
	cd Watermelon-Wars
	```
2. **Install dependencies:**
	```sh
	npm i
	```
3. **Start the development server:**
	```sh
	npm run dev
	```
